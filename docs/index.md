# Stellaris

Stellaris is a market-making platform on Solana. It lets you create automated bots that trade against on-chain liquidity pools, generating volume, injecting liquidity, extracting value, and defending price floors — all from a single dashboard.

## What you can do

- **Manage wallet accounts** — Create HD wallet groups, fund them, transfer tokens, and control access with whitelists
- **Create bots for any pool** — Connect to Pump.fun, Raydium, or Meteora pools in seconds
- **Run trading operations** — Execute Volume, Inject, Extract, and Defend Floor operations with real-time control

## Quick start

1. **Log in** — Connect your Solana wallet (Phantom, Solflare, Torus, or Ledger)
2. **Create an account** — Set a name, wallet password, and save your seed phrase
3. **Create a bot and run your first operation** — Pick a pool, fund your bot, and start a Volume operation

## Key concepts

| Concept | Description |
|---|---|
| **Account** | A named group of HD wallets, protected by a password |
| **Bot** | Connects an on-chain pool to your account's wallets |
| **Trading Operation** | A discrete action (Volume, Inject, Extract, Defend Floor) that runs against a pool |
| **Funding Wallet** | The auto-assigned wallet address you send SOL and tokens to for a bot |
| **Pool** | An on-chain liquidity pool on Pump.fun, Raydium, or Meteora |
